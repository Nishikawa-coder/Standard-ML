6-10.smlについての説明

自由課題２について取り組みました。私は簡単な家計簿を作りました。

<詳細>
ユーザ＝の支出を管理する家計簿です。入力できる種類は衣（日用品の支出等）、食（食費）、住（固定費）、交際費（趣味、遊びを含む）があります。
まずは支出した金額をユーザーに入力してもらいます。
（1000円以上の支出の場合「そんなに使っちゃったんですね…」というコメントが出ます）
その後、衣（日用品の支出等）の場合はcloth、食（食費）の場合はfood、\n住（固定費）の場合はlife、交際費（趣味等）の場合はfunnyを入力してもらいます。
これを繰り返して衣食住交際費のそれぞれの総支出を算出します。
